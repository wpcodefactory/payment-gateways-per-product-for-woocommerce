Show WooCommerce gateway only if there is selected product, product category or product tag in cart.

== Description ==

**Payment Gateways per Products for WooCommerce** plugin lets you show WooCommerce gateway only if there is product of selected **category or tag** in cart. Alternatively you can hide gateway by selected category or tag.

[Payment Gateways per Products for WooCommerce Pro](https://wpfactory.com/item/payment-gateways-per-product-for-woocommerce/) plugin version has options to show/hide the gateways on **per product** basis (i.e. instead of on per product category / tag basis).

== Installation ==

1. Upload the entire plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Start by visiting plugin settings at "WooCommerce > Settings > Payment Gateways per Products".
